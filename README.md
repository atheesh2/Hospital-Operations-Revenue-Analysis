# 🏥 Hospital Operations & Revenue Analysis using SQL

## 📌 Project Overview

This project demonstrates how SQL can be used to analyze hospital operations and financial performance through a relational database.

The database manages patient records, doctor information, appointments, billing, medications, and hospital details.

The objective is to generate business insights that support operational efficiency and revenue analysis.

---

## 🎯 Problem Statement

Hospitals generate large amounts of operational data every day. Without proper analysis, it becomes difficult to monitor revenue, treatment costs, appointment trends, and doctor performance.

This project builds a relational healthcare database and performs SQL analysis to answer key business questions.
